Diseño de Sitios Web - UNAD ECBTI

Informacion general del OVI

Esta plataforma le permitirá adquirir información básica sobre seguridad informática. Se verá información como seguridad informática, mecanismos de seguridad, algunos antivirus y la seguridad informática a través del tiempo.

Objetivos del OVI

De acuerdo al curso escogido para realizar la actividad, en este caso Seguridad Informática, de forma individual se debe llenar el formato OVI donde, habia que poner información como, objetivo general, objetivos especificos y una serie de secciones, donde se habla de informacion especifica : seguridad informática, mecanismos de seguridad, algunos antivirus y la seguridad informática a través del tiempo

Objetivos del curso

Que el estudiante se fundamente teóricamente sobre las generalidades básicas del Diseño Web. Que el estudiante pueda utilizar de forma correcta los lenguajes HTML5 y CSS3 para aplicarlos en la solución de problemas de construcción de Sitios Web. Que el estudiante implemente servicios y sitios web a través de sistemas de control de versiones.

Tematicas del curso

1. Unidad 1 - Introducción al Diseño de Sitios Web: Referentes del Diseño Web. Sistemas de Control de Versiones. Editores de Código Para la Web.

Unidad 2. HTML5: Introducción a HTML5. Estructura básica de HTML5. Aplicación de elementos de HTML5 a un proyecto de diseño Web.

Unidad 3. CSS3: Introducción a CSS3. Estructura básica de CSS3. Aplicación de elementos de CSS3 a un proyecto de diseñoo Web.
